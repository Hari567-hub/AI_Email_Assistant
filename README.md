# AI Email Assistant

An AI-powered email assistant built with Python, Gmail API, and Google Gemini.

## Features

- Read Gmail inbox
- AI email summarization
- Gmail OAuth authentication
- HTML email parsing

## Tech Stack

- Python
- Gmail API
- Google Gemini API
- BeautifulSoup
- OAuth 2.0

## Upcoming Features

- AI reply generation
- Email categorization
- Priority detection
- Streamlit dashboard
- Desktop notifications
